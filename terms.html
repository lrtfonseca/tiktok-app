Terms of Service

This application allows authorized TikTok users to retrieve video metadata and publish video content using the official TikTok API.

Users must authorize the application through TikTok OAuth before any data access or publishing functionality is available.

The application may access video metadata such as video ID, title, publish date and engagement metrics for analytics and content management.

The application will not perform actions on behalf of a user without explicit authorization.

By authorizing the application, the user agrees that the application may interact with the TikTok API to retrieve metadata and publish content on the user's behalf.
